======================================
 An overview about input data formats
======================================

Generally speaking, data formats can be classified into two large
groups:

1. “raw” field data with polar coordinates
2. processed data, with XY(Z) coordinates

The latter are far more easy to process, and currenlty TOPS doesn't
support any polar format, even though work is being done in that
direction.
